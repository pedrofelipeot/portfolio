## 📦 Como executar localmente

1. Clone o repositório:  
```bash
git clone https://github.com/pedrofelipeot/portifolio.git
```

2. Entre na pasta do projeto:  
```bash
cd portifolio
```

3. Instale as dependências:  
```bash
npm install
```

4. Rode o projeto:  
```bash
ng serve
```

5. Acesse no navegador:  
```
http://localhost:4200
```
